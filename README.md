# Urban Heat Island Dashboard for Essex County, New Jersey

This repository contains the full workflow, analysis pipeline, and documentation for the Urban Heat Island Dashboard developed as part of a graduate thesis project. The project integrates remote sensing, geospatial analytics, socioeconomic indicators, and machine learning to support decision-making for climate resilience in Essex County, NJ.

The repository is structured to separate notebooks, source code, sample data, exported outputs, and thesis documentation for clarity and reproducibility.



## Project Objectives

- Extract and map Land Surface Temperature (LST) using satellite data.
- Integrate land cover, traffic intensity, and socioeconomic indicators.
- Develop an interpretable Urban Heat Island (UHI) model.
- Build an interactive geospatial dashboard for policymakers.
- Document the research using transparent, reproducible workflows.
