# 01_notebooks

This folder contains Colab notebooks used for the Urban Heat Island Dashboard project.  
Each notebook corresponds to a major workflow:
- 00_Setup_and_DataCheck.ipynb` - Initial setup and data verification
- 01_LST_Analysis.ipynb  - Land Surface Temperature processing
- 02_LandCover_Processing.ipynb  - Land cover analysis
- 03_Socioeconomic_Analysis.ipynb  - Vulnerability mapping
- 04_RoadTraffic_Analysis.ipynb  - Traffic data integration
- 05_UHI_Modeling.ipynb  - Heat island modeling
- 06_Dashboard_Development.ipynb - Dashboard prototyping



---

## Dashboard Features (Planned)

- UHI hotspot visualization  
- LST trend explorer  
- Land cover and impervious surface overlays  
- Traffic density heatmaps  
- Socioeconomic vulnerability integration  
- Policy scenario comparisons  

---

## Data Sources

- Landsat 8/9 LST
- NLCD Land Cover
- Esri Traffic Counts
- EPA Environmental Justice datasets
- Census ACS data
- Local shapefiles for Essex County municipalities and census tracts

Raw data is kept offline due to size constraints.
